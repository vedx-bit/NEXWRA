## What is AgentCrew?
A Python framework where multiple AI agents collaborate to solve complex problems — one researches, another writes, all automatically.

- 🔬 Market Research Reports
- 💪 Gym Schedule Generator  
- 📝 Content Writer
- 🧠 Custom AI Missions
